# Join the Movement — Design System

A framework-agnostic design system extracted from the Creavue
**“Join the Movement”** marketing website. It ships **design tokens**,
**CSS components**, and **self-hosted fonts** — no framework lock-in.

---

## How to use this Storybook

- **Foundations** — the raw design language: colors, typography, spacing,
  radii, shadows, and motion.
- **Components** — every reusable UI piece with its variants and states.
- **Patterns** — larger compositions (e.g. the hero) assembled from components.

Use the toolbar to switch:

| Control       | Options              | What it does                          |
| ------------- | -------------------- | ------------------------------------- |
| **Theme**     | Dark / Light         | Swaps the full color palette          |
| **Motion**    | On / Off             | Toggles reveal + ambient animations   |
| **CTA style** | Lime / Gradient      | Switches the call-to-action treatment |

---

## Consuming the tokens

```css
/* Everything (fonts + tokens + base + components) */
@import "join-the-movement-design-system/src/styles/index.css";

/* Tokens only */
@import "join-the-movement-design-system/tokens.css";
```

```js
import tokens from "join-the-movement-design-system/tokens.json";
import { colors, typography } from "join-the-movement-design-system/tokens.js";
```

Set the theme on the document root:

```html
<html data-theme="dark" data-motion="on" data-btn="lime">
```
